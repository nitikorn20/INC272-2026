## Step 4: Be Ready for Week 03

The first onsite session is:

| Item | Information |
|------|-------------|
| Week | `Week 03` |
| Date | `20/04/2026` |
| Time | `Monday 09:30` |

Before that session, students should be able to:

- run the mock hardware server
- open the assigned starter applications
- show readiness evidence from Weeks 01-02

If the required workflow is still not ready by the Week 03 check, the readiness task receives `0`.

## Minimum Tools

- Node.js
- Visual Studio Code
- modern browser

You do not need `Proteus` or `Virtual Serial Port Driver` for the default course workflow.
