## Week 01: Online Foundations

**Date:** `06/04/2026`

### Focus

- CSS basics
- JavaScript basics
- local environment preparation

### Student Tasks

- read the course orientation

- verify Node.js installation
- run the assigned HTML, CSS, and JavaScript examples

## Week 02: Online Simulator Familiarization

**Date:** `13/04/2026`

### Focus

- opening the starter web applications

- completing the required readiness preparation

### Student Tasks

- run the mock simulator
- open the assigned starter apps
- confirm expected behavior in the browser
- modify at least one assigned example
- save evidence for the readiness task

## Course Design Principles

- Keep the required setup lightweight.
- Keep the default workflow browser-based and project-oriented.
- Use the mock simulator as the standard course path.
- Keep the exam aligned with the assigned examples and teaching materials.
- Keep the project scope small enough to finish within the onsite weeks.
