## Required Work

### 1. Read the Student Quick Start

Use [Student Quick Start](../docs/student-quick-start.md) to understand the course setup and what to prepare before Week 03.

### 2. Watch the assigned Week 01 videos

Use the Week 01 section in [Legacy Video Map](../docs/legacy-video-map.md).

### 3. Install the required tools

Minimum tools for Week 01:

- Node.js
- Visual Studio Code
- Chrome, Edge, or another modern browser

### 4. Verify Node.js installation

Run:

```bash
node -v
npm -v
```

Expected result:

- both commands print version numbers

### 5. Run the assigned examples

Test at least these files:

- `examples/html/ex01_html_structure/index.html`
- `examples/css/ex03_css_external/index.html`
- `examples/js/ex02_extjs/index.html`

### 6. Make small edits

Make at least one small change in each area:

- HTML
- CSS
- JavaScript

* * *

## Suggested Submission Evidence

- screenshot of `node -v` and `npm -v`
- screenshot of one running HTML, CSS, or JavaScript example
- short note describing what you changed

This evidence supports the Week 03 readiness check.

* * *

## Success Criteria

By the end of Week 01, students should be able to:

- open the repository locally
- confirm Node.js installation
- run the assigned HTML, CSS, and JavaScript examples
- make small edits without browser errors

* * *

## Note

Do not spend this week on advanced tooling. The goal is simple local readiness and browser-side confidence.
